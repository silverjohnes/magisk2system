# Overview

TLDR; This module put files into the write-protected /system dir.
Now it replaces rington in Joying Android 10 headunit with Iphone X's ringtone.

## Installation

Put anything you need into system dir, zip it at normal compression and install in your rooted device with magisk-Manager.

### Russian

Замена рингтона в магнитоле Joying на Abdroid 10. Модуль подходит для подмены любых файлов в /system. Просто положите в папку system внутри архива те файлы, которые вам нужно подбросить, вместо моих, заархивируйте в zip с нормальным сжатием и скормите Маджиску.

Использование модуля на примере замены ирнгтона:
1. Должен быть root (я делал с помощью этого (https://4pda.to/forum/index.php?showtopic=1000996&view=findpost&p=108955404) скрипта просто и быстро). Не забудьте установить и инициировать Magisk, как указано в инструкции.
2. Загружаем архив magisk2system на флешку, архив распаковывать не нужно.
3. В Magisk открываем модули (крайний правый пункт в меню снизу), выбираем Установить с диска и устанавливаем файл из п.2. После установки и перезагрузки у вас будет звонок от Айфона 10.
